# @vis-three/plugin-camera-adaptive

相机自适应窗口改变插件。

安装此插件后，当前的场景相机会自动跟随窗口大小改变而做出调整，不管是在设置窗口大小还是在更换相机都会自适应。

:::warning
如果安装了此插件，又安装了`@vis-three/module-camera`，请不要开启相机自适应配置，这会重复功能。
:::
